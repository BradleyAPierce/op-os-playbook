Dispatcher Paragon is Konica Minolta's premier solution for print management. Developed in partnership with YSoft Corporation, this offering is designed to enhance employee productivity, reduce print-related costs, and ensure document security. It provides central accounting for all print, copy, and scan operations, along with detailed reporting to promote accountability and tools to eliminate waste and encourage responsible behavior. (Also supported on non-Konica Minolta devices with direct support from YSoft).

Dispatcher Paragon is available in two versions: Full License and Essentials.

Dispatcher Paragon Full License:
The Full License includes the entire suite of features outlined below, while Essentials incorporates only key features at a reduced cost. Dispatcher Paragon supports Konica Minolta Card Readers, including the AU-205H and the YSoft MFX Reader.

Dispatcher Paragon Full License offers the following key features:

  • Secure Print Release: Allows users to submit print jobs to any printer within the network and release them wherever needed.
  • Mobile Print: Enables users to send print jobs from their mobile devices via email or a web-release page, providing flexibility for on-the-go employees.
  • Reporting: Tracks and accounts for all copy, fax, print, and scan jobs, detailing the "who/what/where/when" of all print activity.
  • Basic Scanning: Offers Scan to Me, Scan to Home, and Scan to Exchange functionality.
  • Rule-based Printing (RBP): Allows administrators to set individual print conditions for different users.
  • Client-Based Print Roaming (IP Printing): Holds print jobs at the client PC for release at any MFP. Users can still copy, fax, print, and scan even if the server goes down.

Entry-Level Version: Dispatcher Paragon Essentials

Paragon Essentials is an entry-level version available at discounted pricing. It includes authentication, secure pull print, accounting, and basic scanning.

Healthcare Secure Pull Print:
Dispatcher Paragon supports secure pull print workflows within the Epic infrastructure, ensuring document security that complies with HIPAA requirements. Additionally, Paragon is listed in the Epic Connection Hub Showroom.

Dispatcher Suite is an integrated offering of Dispatcher Paragon and Dispatcher Phoenix, providing a new platform that seamlessly integrates single sign-on from print management directly into advanced workflow automation. This suite is ideal for all customers in small-to-midsized businesses (SMBs), as well as Enterprise and Global Major Accounts (GMA). It enables companies and organizations to effectively manage and reduce printing costs, optimize document workflow productivity, and enhance document security.

Dispatcher Suite consists of two modules:

• Dispatcher Paragon: This print management solution by Konica Minolta Inc. (KMI), in partnership with YSoft Corporation, is designed to boost employee productivity, reduce print-related costs, and ensure document security. It offers central accounting for all print, copy, and scan operations, along with detailed reporting to promote accountability and tools to eliminate waste and encourage responsible behavior. (Also supported on non-Konica Minolta devices with direct support from YSoft).

• Dispatcher Phoenix: Konica Minolta's award-winning workflow automation application helps businesses maximize productivity, increase efficiencies, and reduce costs. Dispatcher Phoenix features intuitive MFP integration for document indexing and folder browsing, advanced capture features, automated document processing (such as OCR, forms processing, file routing, Bates stamping, redaction, etc.), and an easy-to-use Workflow Builder Tool featuring LiveFlo technology. (Supported on Konica Minolta devices only).

With this platform, customers can manage their printing and scanning operations with utmost efficiency. This scalable solution offers convenient single sign-on from user authentication to scan workflow applications, an integrated and intuitive user experience at the MFP, powerful user authentication, and easy configuration with a common installer and device synchronization. When sold together on an MFP, discounted pricing will apply to each device license.

Release2Me is Dispatcher Phoenix's secure print release system, which provides staff, managers, administrators, and other employees the ability to control confidential information. With Release2Me, print jobs are held in a queue until they are released at the MFP by an authenticated user. This ensures that documents are not printed until users enter a password or use an ID card to authenticate themselves at the MFP, confirming that they are physically present to pick up their printed documents. Since the print jobs are held in the queue, not on the MFP's Hard Drive, the user can print these documents from any networked device. And unclaimed documents can be deleted from the queue after a specific amount of time.

Secure your printouts and optimize your scan and OCR workflows with Release2Me+. This convenient and secure follow-you printing and scanning solution addresses your document security challenges and scanning needs. With print reporting, print delegation and scan-to-cloud with OCR workflows, Release2Me+ delivers an all-in-one solution for scanning and print release.
