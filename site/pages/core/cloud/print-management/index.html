Dispatcher Stratus is a revolutionary cloud-based document workflow management tool that’s flexible – no footprint, accessible from anywhere, and ready to help streamline your business.  To support businesses in various industries across the globe, Dispatcher Stratus exceeds the standards set forth by HIPAA, GDPR, and more. Dispatcher Stratus combines the use of robotic process automation (RPA), machine learning (ML) and artificial intelligence (AI) to grant organizations the flexibility to implement total process automation where necessary while also enabling task-based workflows that require manual touchpoints. This frees up employees to focus on customers, driving innovation and growing business.

With Dispatcher Stratus’ Release2Me+functionality, organizations can simplify the process of digitizing paper-based documents to email and network or Cloud folder structures. Additionally, the secure print release system included
with Release2Me+ allows your employees to securely print documents, helping to control costs.

Benefits

  • Increasing the security of paper-based documents as they will not be printed until the authenticated user is present at the MFP
  • Reducing the amount of paper, toner, time and energy, thereby minimizing the costs of duplicate and accidental printouts
  • Eliminating the risk of human error during the scanning and filing process via automated document distribution to a variety of Cloud storage application
  • ffering convenient Scan-to-Me, Scan-to-Home and Scan-to-the-Cloud workflows including connectors to Box®, Dropbox®, OneDrive®, OneDrive for Business®, Google, SharePoint®, SharePoint Online®, WebDAV®, FilesAnywhere® and FileAssist
  • Automating data extraction via high-powered OCR. Release2Me+ includes advanced OCR and metadata processing to speed data extraction and eliminate manual data entry
  • Using Print Delegation, you can share print jobs across an organization and enable secure printing even when an employee is absen
  • • Creating transparency into print operations by enabling administrators and other executives to monitor printing activity throughout the organization with Release2Me+ web-based dynamic print reports

Dispatcher Paragon Cloud V2.0 is designed to eliminate the complexities associated with traditional on-premise print infrastructure and deployment headaches. This Software-as-a-Service (SaaS) secure pull print solution simplifies and modernizes print infrastructures by migrating them to the cloud.

Key Features:

  • Unified Print Driver and Queue: Dispatcher Paragon Cloud V2.0 offers a full-featured, single print driver with a single pull print queue for all users across all locations, supporting multiple vendors. This ensures seamless printing regardless of the printer brand.
  • Location-Aware Printing: The new location-aware platform (PC Client required) automatically discovers and installs printers available at the user’s location and removes them when they leave. This feature enhances productivity by allowing users to print immediately without needing IT support.
  • Scan to Cloud Integration: For customers seeking scan-to-cloud apps like Google Drive, OneDrive, and more, Dispatcher Paragon Cloud V2.0 provides seamless user experience with single sign-on (SSO) capabilities to Dispatcher ScanTrip Cloud (purchased separately).

Benefits:

  • Enhanced IT and User Experience: The full-featured single print driver works across all models and brands. Once authenticated at a device, Paragon Cloud V2.0 formats the job, including all finishing options, and sends it to the chosen printer seamlessly.
  • Improved Productivity: Location-awareness allows users to access printers without installing devices or drivers. Available printers are automatically displayed, and those no longer available are removed, reducing post-installation support calls and accommodating the hybrid and mobile workforce.
  • Single Sign-On to Dispatcher ScanTrip Cloud: With SSO, customers can access popular cloud apps like Microsoft OneDrive, Dropbox, and SharePoint Online, along with features like full or zonal OCR, data extraction, intelligent routing based on metadata, and document renaming.

 

Value

  • Multi-Tenant Solution: Multiple customers share a secured server environment.
  • Automated IT Work: Hassle-free printing with immediate end-user fulfillment.
  • Zero Trust Compliance: Designed to meet Zero Trust requirements.
  • Simple Deployment: Easy deployment via Konica Minolta MarketPlace.
  • Flexible Hosting Options: Choose between pure cloud via Intelligent Web Services (IWS)/Microsoft Azure or hybrid hosting with local edge devices.
